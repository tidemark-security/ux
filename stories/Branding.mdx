# Branding

This page is the Storybook starting point for the Tidemark Security UX package. It shows the visual foundations that the component stories build on: palette scales, semantic theme tokens, typography, and surface rules.

## Geometry

The product language avoids soft rounding on primary brand surfaces. When a corner treatment is needed, it should usually be a deliberate bevel rather than a fully rounded card.
